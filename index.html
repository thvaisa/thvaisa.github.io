Work Projects

R2T2

A radiative transfer code which works for dense random media. The R2T2 is a novel method which will be used to simulate light-scattering from asteroid surfaces. Public version coming soon. Tech: Fortran, MPI.

RTCB

A modernized version of the Fortran 77 radiative transfer code RT-CB. The reimplementation worked as a base for the R2T2. Tech: Fortran, MPI and OpenMP.

SIRIS4.1

Fortran and C++ ray tracer which can handle densely packed media. Based on the Fortran SIRIS4. Tech: CGAL

Dense mesh packing

A C++ packing algorithm, which can be used to pack irregular meshes. The output is used as an input for my other works. Tech: FCL: Flexible collision library. Link after published paper.

Contributions to Game Projects

Wankz

Soon to be finished: Wankz is an old-school button masher, in which 16 players can take part in an intergalactic race Wankz. The game was a made at the Tuusula game jam and I implemented the user interface. Tech: Unity3D.

Space station game

A strategy game in which the player builds a space station. The game was made by 3 people as a course project. Tech: Unity3D and Blender.

Solo Game Projects

Rail platformer

WIP: A 2.5D platformer, which is trying to mimic the feeling of games like Pandemonium or Wild 9. Tech used: Unity3D.

VR language learning

Halted WIP: Languages are learned best by using them, even a child can pick a language without reading books. In this game, you learn languages like a child. The player is put into different scenarios from which they need to survive. Everything happens in the foreign language: If a customs officer wants "Huotui", be prepared to show it. Tech used: Unity3D, SteamVR, and Blender (Modeling and animation).
