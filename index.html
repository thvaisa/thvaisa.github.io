Work

R2T2

A radiative transfer code which works for dense random media. The R2T2 is a novel method which will be used to simulate light-scattering from asteroid surfaces. Tech: Fortran, MPI.

RTCB

A modernized version of the Fortran 77 radiative transfer code RT-CB. The reimplementation worked as a base for the R2T2. Tech: Fortran, MPI and OpenMP.

Multiparticle SIRIS

Fortran and C++ ray tracer which can handle densely packed media. Based on the Fortran SIRIS4. Tech: CGAL. Image: Link to the paper and open source version of the

Dense mesh packing

A C++ packing algorithm, which can be used to pack irregular meshes. The output is used as an input for my other works. Tech: FCL: Flexible collision library.

JetCreator for Sispo

Implemented features for the computational space mission camera simulator Sispo. Tech: Python, Cython, C, Docker.

OpenMCT plugins for MCC

Implemented openmct plugins for mission control center. Tech: Python, Javascript, SQL.
